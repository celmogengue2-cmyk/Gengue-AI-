# Gengue AI

**Gengue AI** é uma aplicação criada em Angola, com identidade cultural própria e visão global.  
O objetivo é tornar a inteligência artificial acessível, prática e ligada às realidades locais.

## ✨ Funcionalidades
- Assistente digital com foco em cultura angolana  
- Ferramentas práticas para estudo, trabalho e criatividade  
- Interface simples e acessível para iniciantes
